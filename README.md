# 透明图片查看器 (Transparent Image Viewer)

一个简洁优雅的桌面图片查看工具,支持透明窗口和鼠标穿透功能。

## ✨ 特性

- **透明窗口**: 窗口背景完全透明,图片像浮在桌面上一样
- **拖拽查看**: 直接拖拽图片文件到窗口即可查看
- **自由调整**: 支持调整窗口大小,图片自动适配
- **鼠标穿透**: 可选的鼠标穿透功能,让图片不影响其他操作
- **轻量快速**: 基于现代 Web 技术构建,启动快速,占用资源少

## 🚀 使用方法

1. 安装依赖:
```bash
npm install
```

2. 启动应用:
```bash
npm run dev
```

3. 将图片文件拖拽到窗口中即可查看

## 🛠️ 技术栈

- React + TypeScript
- Electron (桌面应用框架)
- Vite (构建工具)
- Tailwind CSS (样式)

## 📝 适用场景

- 参考图片进行设计工作
- 对照文档进行编程
- 在桌面上固定显示重要信息
- 任何需要图片悬浮显示的场景
