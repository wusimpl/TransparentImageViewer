# 智能图片查看器 (Smart Image Viewer)

一个现代化的 Web 图片浏览器,提供流畅的文件夹浏览和图片预览体验。

## 特性

- **文件夹浏览**: 通过浏览器的文件系统 API 访问本地文件夹,支持子文件夹导航
- **缩略图网格**: 智能加载图片缩略图,支持三种尺寸切换(小/中/大)
- **实时预览**: 点击图片后在右侧预览面板查看大图,支持缩放、拖拽、切换背景
- **智能背景**: 自动分析图片亮度,推荐合适的背景色(深色/浅色/棋盘格)
- **收藏夹功能**: 可将常用文件夹添加到收藏夹,快速访问
- **键盘导航**: 支持方向键切换图片、ESC 关闭预览等快捷操作
- **懒加载优化**: 缩略图按需加载,提升大量图片时的性能

## 使用方法

1. 安装依赖:
```bash
npm install
```

2. 启动开发服务器:
```bash
npm run dev
```

3. 在浏览器中打开显示的本地地址(通常是 http://localhost:5173)

4. 点击左侧边栏的"打开文件夹"按钮,选择包含图片的文件夹

5. 浏览缩略图,点击图片即可在右侧预览

## 技术栈

- **React 18** + TypeScript - 现代化的组件开发
- **Vite** - 极速的构建工具
- **Tailwind CSS** - 实用优先的样式框架
- **Zustand** - 轻量级状态管理
- **Framer Motion** - 流畅的动画效果
- **File System Access API** - 原生文件系统访问

## 支持的图片格式

PNG, JPG, JPEG, WebP, SVG, GIF, AVIF, BMP
